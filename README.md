# Artificial Neural Networks for Disease Prediction  
### Analysis of Depth and Width in Neural Networks

---

## 📌 Project Overview

This project explores how Artificial Neural Networks (ANN) can be used to predict diseases using medical data. The main focus is to understand how changing the depth (number of layers) and width (number of neurons) affects the performance of the model.

During this work, I experimented with different ANN structures and observed how model complexity impacts learning and prediction accuracy.

---

## 🎯 Objectives

- Understand the basics of machine learning and ANN  
- Analyse how depth and width influence model performance  
- Use a real-world dataset for disease prediction  
- Visualise relationships between important medical features  
- Build a simple but meaningful learning-based model  

---

## 📊 Dataset

The dataset used in this project is based on diabetes prediction and includes features such as:

- Pregnancies  
- Glucose level  
- Blood Pressure  
- BMI  
- Age  

The target variable indicates whether a patient has diabetes (1) or not (0).

Dataset Source:  
UCI Machine Learning Repository  
https://archive.ics.uci.edu/ml/datasets/diabetes  

---

## ⚙️ Methodology

The project was carried out in the following steps:

1. Data preprocessing and preparation  
2. Feature and target separation  
3. Model design using ANN  
4. Testing different architectures:
   - Shallow model  
   - Medium model  
   - Deep model  
5. Visual analysis using graphs  

The main goal was not only to build a model but also to understand how it behaves.

---

## 📈 Graphs and Analysis

The following visualisations were created:

- BMI vs Disease Outcome  
- Pregnancies vs Disease Outcome  
- Glucose Distribution  

These graphs helped in understanding how different features contribute to prediction.

---

## 🧠 Key Learnings

From this project, I learned:

- Model structure (depth & width) plays an important role  
- A very simple model underfits  
- A very complex model may overfit  
- A balanced model performs best  
- Features like glucose and BMI are highly influential
